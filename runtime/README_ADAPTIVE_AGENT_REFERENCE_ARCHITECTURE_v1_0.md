# Adaptive Agent Reference Runtime v1.0

The Python files in this directory are an executable reference model.

- `kuuos_adaptive_agent_reference_types_v1_0.py` records the current planes, stages, recovery algebra, and implementation map.
- `kuuos_adaptive_agent_runtime_megamodel_v1_0.py` records the current runtime-model inventory and relation set.
- `kuuos_adaptive_agent_transition_kernel_v1_0.py` implements the current global transition function and state checks.
- `kuuos_adaptive_agent_reference_scenarios_v1_0.py` supplies nominal and fault scenarios.

This model is descriptive and may be revised, extended, or replaced as KuuOS develops.
