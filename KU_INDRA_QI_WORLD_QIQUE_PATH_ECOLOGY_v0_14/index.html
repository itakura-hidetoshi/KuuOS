    Kū–Indra Qi WORLD Qi-QUE Path Ecology v0.14¶

    Purpose¶

    v0.14 lifts the existing daemon Qi-QUE diagnostic from a compact Process Tensor summary to the Indra–Qi noncommutative Mandala WORLD model. It does not alter the WORLD model or overwrite the v0.13 governance decision. It adds a digest-bound, read-only ecology review that asks whether an apparently stable generational result has retained enough path diversity, branch accessibility, and multi-WORLD non-collapse to support a bounded promotion recommendation.

    Qi-QUE remains a derived gauge. It is not Qi as substance, a final ontology, a truth authority, or a WORLD update command.

    Inputs¶

    The runtime binds exactly:

    • ku_indra_qi_noncommutative_mandala_world_state.json;
    • indra_qi_generational_governance_state_v0_13.json;
    • indra_qi_generational_governance_recommendation_v0_13.json;
    • a v0.14 plan and license bound to all three source digests.

    The source files are read-only. A repeated review-run ID or repeated pair of WORLD-state and v0.13 recommendation digests is rejected.

    WORLD Qi-QUE observables¶

    v0.14 computes:

    • patch_participation_ratio: normalized participation of local WORLD patches;
    • dominant_patch_share: concentration in the most active patch;
    • flow_concentration: concentration in a single Qi flow channel;
    • connection_utilization: fraction of declared IndraNet connections carrying flow;
    • overdiffusion_score: broad patch participation without adequate transport structure;
    • branch_energy: combined topological branching and normalized flow entropy;
    • loop_mode_lock: flow captured by the dominant holonomy cycle;
    • scar_reentry_score: holonomy lock retained through visible transport residue;
    • multi_world_diversity: patch participation and branch energy under Mandala non-collapse;
    • false_stability_pressure: weighted localization, branch loss, loop lock, scar reentry, and diversity loss.

    The observables are conventional-side diagnostic projections. They do not become facts merely because their checker passes.

    Licensed localization¶

    Localization is not automatically pathological. A plan may declare a bounded set of licensed_localization_patch_ids. If the dominant patch belongs to that set, the localization component receives explicit bounded relief. This does not waive branch-energy, holonomy-lock, scar, diversity, overdiffusion, or false-stability gates.

    Thus protective concentration can remain possible without converting all concentration into unrestricted authority.

    Relation to v0.13¶

    v0.14 supplements, but never rewrites, v0.13:

    v0.13 promote_bounded
  + WORLD Qi-QUE ecology gates pass
  -> ecology_compatible_bounded_promotion

v0.13 promote_bounded
  + localization / branch / diversity / scar gate fails
  -> reopen_branches_recommended

v0.13 promote_bounded
  + overdiffusion gate fails
  -> focus_or_reobserve_recommended

v0.13 rollback or quarantine
  -> preserve the same recommendation class

    A low observation debt, high recoverability reserve, low intervention residue, and successful replay are therefore not sufficient by themselves. v0.14 can identify false stability when those values coexist with path collapse.

    Decisions¶

    The runtime emits exactly one recommendation:

    • hold_for_observation;
    • ecology_compatible_bounded_promotion;
    • reopen_branches_recommended;
    • focus_or_reobserve_recommended;
    • rollback_recommended;
    • quarantine_recommended.

    All are recommendations only. No decision directly executes a generation, changes an operator algebra, changes an IndraNet connection, mutates a causal variable, or actuates the external world.

    Outputs¶

    indra_qi_world_qique_path_ecology_state_v0_14.json
indra_qi_world_qique_path_ecology_recommendation_v0_14.json
indra_qi_world_qique_path_ecology_ledger_v0_14.jsonl
indra_qi_world_qique_path_ecology_receipt_v0_14.json
indra_qi_world_qique_path_ecology_audit_v0_14.jsonl

    Authority boundary¶

    v0.14 preserves:

    • Qi as gauge-covariant relational effective flow, not substance;
    • multi-WORLD non-collapse;
    • visible holonomy and transport scar;
    • non-Markov and Process Tensor provenance;
    • candidate weighting as non-truth;
    • strict separation of recommendation, execution, WORLD update, actuation, and truth authority.
