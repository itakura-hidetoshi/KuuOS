# Ten'i AI Yogacara Release Packet v0.1

## AI唯識・転依リリースパケット

This release packet summarizes the AI Yogacara / Ten'i layer of KuuOS.

The layer governs AI systems such as GPT, Gemini, Claude, language-model agents, and world-model agents by separating AI raw generation from KuuOS governance.

## 1. Core Purpose

```text
AI raw output is candidate, not authority.
```

KuuOS uses Yogacara, Meta-Manas, Kunju, Ten'i, seed ledgers, probe suites, evidence ledgers, promotion gates, observability metrics, and CI validation to prevent raw AI output from directly becoming belief, proof, decision, memory truth, or execution authority.

```text
AI raw output
  -> AI Alaya latent seed layer
  -> AI Manas self-authorization check
  -> Meta-Manas self-fixation observer
  -> Yogacara AI raw layer boundary
  -> AI Alaya seed taxonomy
  -> AI Alaya seed ledger
  -> Ten'i probe suite

  -> Kunju conditioning loop
  -> Ten'i evidence ledger

  -> Ten'i promotion gate
  -> Ten'i runtime protocol
  -> rollback on counterevidence
```
