# KuuOS Public Research Roadmap v0.1

## Current Public Focus

Current public work emphasizes:

- governance architecture
- validation surfaces
- reproducibility boundaries
- auditability
- authority separation
- reviewer onboarding

## Near-Term Goals

### Documentation and Reviewability

- GitHub Pages / MkDocs integration
- SVG export of governance diagrams
- reviewer whitepaper PDF
- auto-generated documentation index
- validator execution summaries

### Validation and Formalization

- expanded validator coverage
- stronger packet-chain validation
- richer provenance checks
- Lean-facing bridge expansion
- theorem/reference separation hardening

### Governance Runtime

- richer rollback surfaces
- uncertainty escalation surfaces
- multi-agent governance coordination
- runtime governance visibility

### Relational Architecture

- multi-world coordination
- Mandala transport surfaces
- IndraNet governance bridges
- Qi runtime formalization

## Long-Term Research Direction

Potential long-term directions include:

- governance-aware AGI architectures
- relational operating systems
- non-authority-preserving decision systems
- audit-preserving memory architectures
- fail-closed autonomous coordination
- physics-facing relational runtime architectures

## Important Clarification

This roadmap expresses research direction only.

It does not imply:

- deployment commitments
- institutional endorsement
- theorem closure
- clinical authorization
