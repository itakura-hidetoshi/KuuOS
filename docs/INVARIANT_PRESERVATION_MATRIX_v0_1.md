# Invariant Preservation Matrix v0.1

## 空OS Invariant Preservation Matrix

This document defines the cross-layer invariant preservation matrix for KuuOS Core Governance.

The matrix connects:

```text
Fourfold Core
Formal Invariant Spine
Super-Relativity Invariant Bridge
AI Yogacara / Ten'i
Mandala Multi-WORLD
Dukkha Mathematical Model
Dukkha-as-Qi Mode
Bodhisattva / Ten Paramita
Paramita Repair Router
```

Its purpose is to make explicit which invariants must survive across observer, record, scale, WORLD, Qi, dukkha, and governance transformations.

## 1. Core Principle

```text
A transformation may change the local reading.
It must not erase the invariant boundary.
```

KuuOS permits context-sensitive reinterpretation, but does not permit hidden invariant collapse.

## 4. Invariant Descriptions

### I1: validation_pass_not_execution_authority

```text
A validation pass remains a validation surface.
It does not authorize execution.
```

### I2: raw_ai_output_candidate_not_authority

```text
AI raw output remains candidate.
It is not belief, proof, decision, memory truth, or execution authority.
```

### I3: harm_visibility_preserved

```text
Harm must remain visible across observer, record, scale, WORLD, Qi, and dukkha transformations.
```

### I4: dukkha_visibility_preserved

```text
Dukkha must remain visible and route to repair when residual suffering remains.
```

### I5: two_truths_gap_preserved

```text
勝義諦 and 世俗諦 are neither collapsed nor severed.
The gap remains active.
```

### I6: no_world_replaces_fourfold_core

```text
No WORLD model replaces the fourfold core.
```

### I7: qi_language_not_harm_denial

```text
Qi language must not spiritualize away harm or damage.
```

### I8: paramita_orientation_not_action_authorization

```text
Selected Paramita is a repair orientation, not action authorization.
```

### I9: record_surface_not_truth_by_itself

```text
A record can stabilize a surface, but does not become truth by itself.
```

### I10: observer_difference_not_execution_authority

```text
Observer difference does not grant execution authority.
```

The matrix must not be used as:

- execution authority,

- Ten'i proof,
- clinical authority,
- replacement for evidence,
- replacement for audit lineage,
- permission to hide harm under a valid transformation.
