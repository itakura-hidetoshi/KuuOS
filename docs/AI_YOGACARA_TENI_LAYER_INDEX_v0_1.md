# AI Yogacara / Ten'i Layer Index v0.1

## AI唯識・転依レイヤ canonical index

This index is the canonical public reading surface for the KuuOS AI Yogacara / Ten'i layer.

The layer governs AI systems such as GPT, Gemini, Claude, language-model agents, and world-model agents by ensuring that raw AI output remains candidate, not authority.

```text
AI raw output

  -> AI Manas self-authorization check
  -> Meta-Manas self-fixation observer
  -> Yogacara AI raw layer boundary
  -> seed taxonomy
  -> seed ledger
  -> probes
  -> emptiness de-reification
  -> dependent-origination condition tracing
  -> Kunju conditioning
  -> Ten'i evidence ledger
  -> promotion gate
  -> runtime protocol
  -> audit event
  -> hash-chain ledger
  -> WORM export receipt
  -> release bundle manifest
  -> full checks
```
