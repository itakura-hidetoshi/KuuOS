It is an append-only index for the KuuOS AI Yogacara / Ten'i layer.

## 4. Validation Coverage

The validation cases cover:

```text
single correction is not Ten'i
MemoryOS update is not Ten'i
provisional Ten'i requires repeated early evidence
confirmed Ten'i requires durable repeated evidence
interface-level shift is not model-level Ten'i
counterevidence can trigger rollback
seed ledger is not model identity
metrics are not execution authority
```

## 5. Fixed Points

```text
Ten'i must be observable before promotion.
Ten'i status is not execution authority.
Metrics are observability surfaces, not authority.
A seed entry is tendency evidence, not model essence.
Control surface scope limits Ten'i claim scope.
Future counterevidence may rollback a Ten'i promotion.
```
