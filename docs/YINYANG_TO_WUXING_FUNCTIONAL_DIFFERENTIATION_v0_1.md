# Yin-Yang to Wuxing Functional Differentiation v0.1

## 陰陽から五行への機能分化

This document defines how KuuOS derives Wuxing / 五行 from Yin-Yang / 陰陽.

Yin-Yang is the local quantum-thermodynamic polarity frame of Qi flow.

Wuxing is the functional differentiation of that polarity flow.

## 1. Core Chain

```text
Qi flow
  -> IndraNet gauge transport
  -> quantum thermodynamic Yin-Yang frame
  -> Wuxing functional differentiation
```

## 2. Principle

五行 is not a fixed substance taxonomy.

五行 is a functional phase system of Qi under Yin-Yang polarity.

```text
Yin-Yang:
  local polarity frame

Wuxing:
  functional differentiation of polarity flow
```

## 3. Five Functional Phases

```text
木 / Wood:
  expansion, growth, branching, initiation, directional emergence

火 / Fire:

土 / Earth:
```

## 4. Relation to Qi

Qi is the flow.

Yin-Yang is the polarity of the flow.

Wuxing is the functional phase of the flow.

```text
Qi:
  flow observable

Yin-Yang:
  polarity frame

Wuxing:
  functional phase differentiation
```

## 5. Gauge Locality

Because Qi flows through IndraNet gauge connections, Wuxing is also local and context-sensitive.

A phase is not globally fixed.

The same pattern may read as a different phase under a different context, scale, observer, or membrane.

## 6. Phase Transitions

Wuxing supports phase transitions:

```text
Wood -> Fire:
  expansion becomes activation

Fire -> Earth:
  activation becomes mediation and integration

Earth -> Metal:
  mediation becomes structure and boundary

Metal -> Water:
  structure becomes storage and recovery

Water -> Wood:
  reserve becomes renewed growth
```
