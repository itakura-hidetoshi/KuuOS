## 1. Purpose

This document fixes the Qi Process Tensor as the conventional-truth temporal
substrate for KuuOS autonomous AI.

The purpose is not to grant execution authority.  The purpose is to give KuuOS a
non-Markovian temporal body: a multi-time, intervention-sensitive,
memory-preserving process structure on which Qi flows as history-bearing
relational action.

Canonical statement:

```text
Qi Process Tensor is the conventional-truth temporal substrate of KuuOS.
Qi is the history-bearing relational/action flow on that substrate.
Autonomy means history-aware candidate generation under safety gates, not
ungated self-execution.
```

## 2. Two-truth placement

The Qi Process Tensor belongs to conventional truth.

```text
Ultimate side:
  emptiness, non-reification, non-self-subsisting authority

Conventional side:
  multi-time process structure, observation, intervention, memory, residue,
  recoverability, and candidate action flow
```

ontology, proof authority, or self-grounding substance.

KuuOS autonomy is a gated loop:

```text

-> remember
-> reflect
-> plan
-> gate
-> propose/intervene-if-licensed
-> update process history
```

Module mapping:


The loop is autonomous in candidate generation and history integration, but not
in authority.  Execution remains gated.
