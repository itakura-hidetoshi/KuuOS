# Governance Release Checklist v0.1

## 空OS Governance Release Checklist

This checklist defines the release-preparation procedure for public KuuOS Core governance changes.

It is an operational surface, not authority.

## 1. Required Command

Before release or merge, run:

```bash
make all-governance-checks
```

or:

```bash
python3 scripts/run_all_governance_full_checks_v0_1.py
```

Expected result:

```text
PASS: KuuOS all governance full checks completed
```
