## Newly implemented

v0.23 integrates ObserveOS, VerifyOS, BeliefOS, PlanOS, MemoryOS, and the Qi process tensor into a bounded cognitive-memory and credit-assignment layer.

```text
observed effect
→ bounded verification
→ local/plural belief interpretation
→ non-Markov memory consolidation
→ bounded non-causal step credit
→ future-only belief and plan candidates
```

## Current classification

