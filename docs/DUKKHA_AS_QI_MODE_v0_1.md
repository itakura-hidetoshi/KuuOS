# Dukkha as Qi Mode v0.1

## 苦は気の一つの相である

This document integrates the KuuOS Dukkha Mathematical Model with the Samvrti Qi Layer.

In KuuOS, dukkha / 苦 is not separate from Qi / 気. Dukkha is a distorted, blocked, overbound, scarred, or mis-transported mode of Qi on the conventional surface / 世俗諦.

Qi is the readable effective flow pattern generated from dependent-origination vibration. Dukkha appears when that flow loses recoverable circulation, becomes fixed by attachment, collapses in a WORLD-reading chart, is scarred by memory, is overclaimed by authority, or fails to glue / transport across worlds.

## 1. Core Principle

```text
Dukkha is a mode of Qi, not a separate substance.
```

More precisely:

```text
Dukkha = Qi under obstruction, fixation, collapse, scar, authority overreach, gluing defect, or transport defect.
```

This preserves both:

```text
Qi is not a fixed substance.
Dukkha is not erased by calling it Qi.
```

## 4. Harm Rule

Calling dukkha a Qi mode must not hide harm.

```text
Dukkha-as-Qi does not dissolve E_harm.
```

If E_harm is present, it remains visible as a repair target.

```text
harm-bearing Qi mode
  -> harm_visible
  -> repair route
```

## 5. Yin-Yang Reading

Dukkha-Qi can appear through Yin-Yang imbalance or phase failure:

```text
excess Yang:
  agitation, overactivation, authority overreach, forced execution

excess Yin:
  stagnation, collapse, withdrawal, non-response

Yin-Yang disconnection:
  chart collapse, transport mismatch, inability to complete circulation

blocked transition:
  gluing defect, membrane mismatch, unresolved obstruction
```

This does not reduce Yin-Yang to pathology. It only defines the pathological Qi mode when flow is obstructed.
